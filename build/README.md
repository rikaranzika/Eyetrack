# Icon Files

This directory contains the application icon for the Eye Tracking Controller.

## Icon Requirements

For Windows builds, the icon should be:
- Format: `.ico` file
- Sizes: 256x256, 128x128, 64x64, 48x48, 32x32, 16x16 pixels
- Named: `icon.ico`

## Creating an Icon

You can create an icon using online tools or image editors:
1. Use a tool like https://icoconvert.com/ to convert PNG to ICO
2. Or use GIMP, Photoshop, or other image editors
3. The icon should represent eye tracking (e.g., eye symbol, camera, etc.)

## Placeholder

If no custom icon is provided, the default Electron icon will be used.
To add a custom icon, simply place `icon.ico` in this directory.
